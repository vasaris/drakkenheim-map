# drakkenheim-map

Интерактивная карта **города Драккенхейм** для мульти-GM кампании West Marches
(D&D 5e 2014, вселенная Drakkenheim). Туман войны («Дымка»), статусы зон, двойной текст
(для игроков / для мастера). Терминология — из World Codex (9 зон, пояса 1–13).

Деплой: GitHub Pages → **https://vasaris.github.io/drakkenheim-map/**

## ⚠️ Копирайт — жёсткое правило

Репозиторий публичный, поэтому **официальная карта и арт из книг сюда не коммитятся
никогда**. Картинка карты грузится в рантайме через file-picker и кэшируется в браузере
(IndexedDB; в репо её нет). `assets/` и форматы картинок/PDF — в `.gitignore` (папка держится
через `.gitkeep`). Извлечение из книжных PDF — только локально.

## Как пользоваться

- **Публичный сайт = вьюер.** Видны только зоны со статусом ≠ «скрыто» и игроцкие маркеры;
  всё неоткрытое — под Дымкой (туман войны). Каждый открывает карту своей книги через
  «Загрузить карту» (кэшируется локально в IndexedDB, на чужие устройства не уходит).
- **Редактор = локально**, по флагу `?edit=1` (напр. `index.html?edit=1`). Даёт мастер-
  инструменты: рисовать/править зоны, ставить маркеры, менять статусы и тексты, **экспорт**
  `data/zones.json` / `data/markers.json` для коммита.
  **`?edit=1` — локальный режим автора, не контроль доступа** (любой может его дописать) — поэтому
  реальная защита секретов держится на **шифровании `gmText`**: чтобы видеть/править GM-заметки,
  в панели автора введи **пароль мастера** («Расшифровать GM»); при экспорте `gmText` пере-шифровывается.
- **Локальный запуск редактора:** данные грузятся через `fetch`, поэтому открывать нужно по
  http, а не `file://`. Из корня репо: `python3 -m http.server` → `http://localhost:8000/?edit=1`.

## Структура данных

- `data/zones.json` — 9 зон Кодекса: `id, name, band, owner, status, playerText, gmText, polygon`.
  **`polygon` — нормализованные координаты `[x,y]` в долях 0..1** от натуральных размеров карты
  (переживают ужатие/смену разрешения; рендер домножает на ширину/высоту картинки).
- `data/markers.json` — маркеры (`type`: location / faction / danger / secret / **hub**).
  Сейчас два: **Эмбервуд** (hub, off-map у южной кромки) и **Парк Королевы** (danger, глубокая Дымка).
- `status` зоны/маркера: `hidden` (плотная Дымка) · `known` (редеет) · `scouted` (тонкая) · `explored` (чисто).
- **`gmText` зашифрован** (AES-GCM, ключ из пароля мастера через PBKDF2, общий salt + IV на поле).
  В публичном `data/*.json` хранится только шифр `{enc, salt, iv, ct}` (base64); вьюер его не читает
  и не показывает. **Теперь в `gmText` можно класть реальные секреты кампании** (сюжетные повороты,
  тайны NPC, развязки) — наружу уходит лишь шифр. Пароль вводится в режиме автора в рантайме и
  нигде не хранится (ни в коде, ни в localStorage/IndexedDB). Открытые поля
  (`name, visibleName, playerText, band, owner, status, polygon, координаты, type`) не шифруются.

## Дорожная карта

**Финальный движок v1 (готово):** тематическая **Дымка-туман** (аметист, 4 плотности, фетеренные
края, медленный дрейф, плавный reveal, `prefers-reduced-motion`); **IndexedDB**-кэш полноразмерной
карты; **шифрование GM-слоя** (AES-GCM + PBKDF2, пароль только в рантайме, в репо лишь шифр).

**v2 — вложенные слои карт** (извлечение тем же конвейером, что и Фаза 1; пока не строим):
континентальный обзор «The Continent» (SCGD ~стр.12) → зум в **Вестемар** (~стр.41; там
**Альтберг** — стартовый город каравана Эрена Марлоу) → **Drannsmund** (стр.61: ближние
окрестности — Эмбервуд, дороги, маршрут) → текущая городская карта. Региональные поселения
(Альтберг и пр.) — маркеры overworld-слоя v2, не зоны.
Канон-маршрут стартера: **Альтберг → (3 недели) → Эмбервуд (≈5 миль южнее) → Драккенхейм.**

Возможный полиш: кастомные SVG-иконки маркеров; готик-оболочка (Forum + Spectral, золотые
волоски, Кратер-сигнатура); редактор зон с drag вершин и undo.

## Заметка по синку текстов

`playerText/gmText` зон приведены к **реальной геометрии карты** (карта повёрнута относительно
cardinal-описаний Кодекса: old_town=запад у Замка, spokes=СВ, sprawl=юг, middle/temple=юг-центр,
kingside=верх-слева у King's Gate). **Кодекс в Notion** ещё несёт старые стороны света — отдельный
долг такта codex-sync (внести те же правки + факт поворота карты).
