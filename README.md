# drakkenheim-map

Интерактивная карта **города Драккенхейм** для мульти-GM кампании West Marches
(D&D 5e 2014, вселенная Drakkenheim). Leaflet-движок поверх тайл-пирамиды квадратного
русского мастер-растра. Туман войны («Дымка»), статусы зон, двойной текст (для игроков /
для мастера). Терминология — из World Codex (10 зон, пояса 1–13).

## ⚠️ Копирайт — жёсткое правило

Репозиторий публичный, поэтому **официальная карта и арт из книг сюда не коммитятся
никогда**. Тайл-пирамида и мастер-растр карты живут в приватном хосте
(`~/Downloads/drakkenheim_maphost`) и в этот репозиторий не входят — локально `tiles/`
подключается симлинком на `drakkenheim_maphost/tiles_v3` (см. `tests/global-setup.js`).

## Как пользоваться

- **Публичный сайт = вьюер.** Видны только зоны со статусом ≠ «скрыто» и игроцкие маркеры;
  всё неоткрытое — под Дымкой (туман войны).
- **Мастер-режим** — кнопка «Мастер» в шапке (не флаг URL). Пароль мастера расшифровывает
  GM-заметки и включает авторские инструменты: рисовать/править зоны, ставить маркеры,
  менять статусы и тексты, **экспорт** `data/v3/zones.json` / `data/v3/markers.json` для
  ручного коммита. Пароль нигде не сохраняется — только в памяти вкладки на сессию.
- **Локальный запуск:** данные и тайлы грузятся через `fetch`, поэтому открывать нужно по
  http, а не `file://`. Из корня репо: `python3 -m http.server` → `http://localhost:8000/`.

## Структура данных

- `data/v3/zones.json` — зоны Кодекса: `id, name, band, owner, status, playerText, gmText, polygon`.
  **`polygon` — нормализованные координаты `[x,y]` в долях 0..1** от мастер-растра
  (переживают смену разрешения/тайлинга; рендер домножает на размер тайл-пирамиды).
- `data/v3/markers.json` — маркеры (`type`: location / faction / danger / secret / **hub**).
- `status` зоны/маркера: `hidden` (плотная Дымка) · `known` (редеет) · `scouted` (тонкая) · `explored` (чисто).
- **`gmText` зашифрован** (AES-GCM, ключ из пароля мастера через PBKDF2, AAD = id объекта).
  В публичном `data/v3/*.json` хранится только шифр `{enc, v:2, salt, iv, ct}` (base64) —
  открытый текст наружу не уходит. Открытые поля (`name, visibleName, playerText, band,
  owner, status, polygon/координаты, type`) не шифруются.

## Деплой

Тайл-пирамида и мастер-растр — приватный хост (`drakkenheim_maphost`), не в этом
репозитории. Деплой — вручную, `drakkenheim_maphost/scripts/deploy.sh [--yes]`: собирает
app shell (этот репозиторий: `index.html`, `css/`, `js/`, `data/`) + `tiles_v3` во временный
каталог, гоняет secrets-гейт (`scripts/check-secrets.sh`), аплоадит на Cloudflare Pages
(`wrangler pages deploy`, проект `drakkenheim-map`). Прод — за Cloudflare Access.
